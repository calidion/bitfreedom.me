# bitfreedom.me
我的比特币自由交易之梦

## 导论

比特币自从横空出世，极大的影响了人类社会的货币演进，从而影响了政治格局。

让人类的财富交易方式
从纸币切换成了数字货币，
不但形式发生重大变化，就连存储的形式也完全变化了。

## 比特币试图要解决的问题

1. 货币发行权的垄断问题
2. 纸币的超发造成对其它人的财富的略夺的问题
3. 货币发行机构的信任问题
4. 货币生成的无规则问题

## 比特币存在的问题

1. 合法性问题不受一些国家支持
2. 容易受金融大鄂的影响
3. 交易所未去中心化
4. 无法实现与实物的交易

## 这个项目的目标

1. 实现去中心化、去信任、去运营的交易所
2. 解决与实物的直接交易问题
